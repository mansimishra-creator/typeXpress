# typeXpress
TypeXpress: Typing Speed & Accuracy Analyzer

Project Goal
 -> The primary goal of TypeXpress is to provide a standardized, stress-free way for individuals to measure their typing proficiency. We focused on creating a user-friendly application that gives instant feedback without the clutter of existing tools

 Key Features
  ->Fair Testing: Uses short, single-sentence typing tests to avoid pressure and frustration from long paragraphs
  ->Difficulty Selection: Offers Easy, Medium, and Hard difficulty levels suitable for all typing levels
  ->Accurate Measurement: Real-time calculation of key metrics including Words Per Minute (WPM), accuracy percentage, and total time taken.
  ->Instant Feedback: Provides immediate, color-coded output (Green for correct, Red for incorrect or missing characters) for quick understanding of performance.
  ->Progress Tracking: Stores results (name, level, speed, accuracy) in a leaderboard.csv file for easy comparison and tracking growth.
